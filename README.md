# 🛡️ SevaSyncAI: Tactical Command & Mission Control

**SevaSyncAI** is a high-fidelity, agentic mission control platform designed for real-time crisis management, resource logistics, and community synchronization. Built with a "Command Center" aesthetic, it provides mission commanders with the tools needed to triage emergencies, track field agents, and optimize resource distribution.

## 🚀 Key Modules

- **📡 Strategic Dashboard**: Real-time map intelligence with live asset tracking and crisis telemetry.
- **🛡️ Mission Matrix**: Tactical triage of field reports and autonomous dispatch protocols.
- **📦 Logistics Hub**: High-precision inventory management for critical resources (Oxygen, Food, Medical).
- **🩸 Blood Command**: Real-time donor matrix and inventory synchronization.
- **👥 Personnel Matrix**: Agent directory with dynamic access control and audit logging.
- **🔔 Alerts Hub**: Intelligence stream with live telemetry monitoring and mission briefings.

## 🛠️ Technology Stack

- **Frontend**: React, Vite, Framer Motion (Animations), Lucide React (Icons), Leaflet (Maps).
- **Backend**: Python, Flask, SQLite (Tactical Data persistence).
- **Styling**: Vanilla CSS, Modern Design System with Glassmorphism.

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   ```

2. **Frontend Setup**:
   ```bash
   cd sevasync-dashboard
   npm install
   npm run dev
   ```

3. **Backend Setup**:
   ```bash
   cd backend
   pip install -r requirements.txt
   python main.py
   ```

## 🌍 Deployment

- **Frontend**: Optimized for [Vercel](https://vercel.com).
- **Backend**: Recommended for [Render](https://render.com) or [Railway](https://railway.app) for data persistence.
